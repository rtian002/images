# 图片访问使用github加速代理
访问地址：
```
#https://raw.githubusercontent.com/用户名/仓库/分支/文件名
https://raw.githubusercontent.com/rtian002/images/main/wukong.jpg
```

```
# 将 raw.githubusercontent.com 替换为 raw.gitmirror.com
https://raw.gitmirror.com/rtian002/images/main/wukong.jpg
```
```
#https://cdn.gitmirror.com
# GitHub /gh/:user/:repo/:tag/:file
https://cdn.gitmirror.com/gh/rtian002/images/main/wukong.jpg
```

```
# releases文件链接前加https://hub.gitmirror.com/
# typecho博客程序（已展开，解压即用；内含first.sh初始化配置程序）
https://hub.gitmirror.com/https://github.com/rtian002/images/releases/download/typecho/typecho.tar.gz
```
